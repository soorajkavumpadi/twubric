Project : Twrubrics
Framework: vue.js 
About
The web app is build using vue.js and used multiple packages to support its functionality such as lodash, moment etc. Boostrap is used incase of ui to enhance its ui.

Keyboard shortcuts
all the sorting can be perfromed with keyboard shorcuts: 
alt + t => sort based on twrubrics score
alt + f => based on friends
alt + c => based on chirpiness
alt + i => based on influence
